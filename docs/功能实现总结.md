## 一、新增文件清单

### 1. 模型层 (pkg/model/)

#### combat_rules.go - 战斗规则相关的数据结构
**文件路径**: `pkg/model/combat_rules.go`  
**代码行数**: ~75行  
**主要功能**:
- `CoverType` - 掩护类型枚举(None/Half/ThreeQuarters/Total)
- `CoverBonus` - 掩护加值结构(AC加值和DEX豁免加值)
- `GrappleState` - 擒抱状态结构
- `JumpType` - 跳跃类型枚举(Long/High)
- `SuffocationState` - 窒息状态结构

### 2. 规则层 (pkg/rules/)

#### combat_rules.go - 战斗规则实现
**文件路径**: `pkg/rules/combat_rules.go`  
**代码行数**: ~650行  
**主要功能**:

##### 借机攻击 (Opportunity Attack) - PHB Ch.9
- `CanTakeOpportunityAttack()` - 检查是否能进行借机攻击
- `IsWithinReach()` - 检查目标是否在触及范围内
- `CalculateGridDistance()` - 计算网格距离(5-10-5规则)

##### 双持武器 (Two-Weapon Fighting) - PHB Ch.9
- `CanUseTwoWeaponFighting()` - 检查是否能使用双持武器
- `CalculateOffHandDamage()` - 计算副手攻击伤害(不加属性修正)

##### 擒抱 (Grapple) - PHB Ch.9
- `PerformGrapple()` - 执行擒抱(力量运动对抗检定)
- `CanGrapple()` - 检查是否能擒抱目标(体型限制)
- `EscapeGrapple()` - 尝试逃脱擒抱

##### 推撞 (Shove) - PHB Ch.9
- `PerformShove()` - 执行推撞(推倒或推开5尺)
- `CanShove()` - 检查是否能推撞目标

##### 掩护系统 (Cover) - PHB Ch.9
- `CalculateACWithCover()` - 计算包含掩护的AC
  - 半身掩护: +2 AC
  - 四分之三掩护: +5 AC
  - 全身掩护: 无法被直接攻击
- `CalculateDexSaveWithCover()` - 计算包含掩护的敏捷豁免
- `CanTargetWithCover()` - 检查是否能攻击有掩护的目标

##### 跳跃系统 (Jumping) - PHB Ch.8
- `CalculateJumpDistance()` - 计算跳跃距离
  - 跳远: 有助跑=力量值尺数,立定=力量值/2
  - 跳高: 有助跑=3+力量调整值尺数,立定=一半

##### 坠落伤害 (Falling) - PHB Ch.8
- `CalculateFallDamage()` - 计算坠落伤害
  - 每10尺1d6伤害
  - 最多20d6

##### 窒息规则 (Suffocation) - PHB Ch.8
- `CalculateHoldBreathTime()` - 计算闭气时间
  - 1 + 体质调整值(分钟),最少30秒
- `CalculateSuffocationRounds()` - 计算窒息轮数
  - 体质调整值(最少1轮)

##### 专注豁免机制 (Concentration Saves) - PHB Ch.10
- `CalculateConcentrationDC()` - 计算专注豁免DC
  - DC = 10 或 伤害/2,取较高值
- `PerformConcentrationSave()` - 执行专注豁免(体质豁免)

##### 团体检定 (Group Checks) - PHB Ch.7
- `PerformGroupCheck()` - 执行团体检定
  - 至少一半成员成功则团体成功

##### 合作检定 (Working Together) - PHB Ch.7
- `PerformWorkingTogether()` - 执行合作检定
  - 领导者进行检定并具有优势

##### 暴击规则 (Critical Hits) - PHB Ch.9
- `CalculateCriticalDamage()` - 计算暴击伤害(伤害骰x2+修正)
- `IsCriticalHit()` - 检查是否是暴击(自然20)
- `IsCriticalFumble()` - 检查是否是大失败(自然1)

## 四、实现细节说明

### 1. 借机攻击规则
```go
// 当一个你能看到的敌对生物离开你的触及范围时,你可以用反应对其进行一次近战攻击
// 触及范围: 默认5尺,长触及武器10尺
// 条件:
// - 攻击者不能失能
// - 目标必须在触及范围内
// - 目标正在离开触及范围
// - 攻击者有可用的反应
```

### 2. 双持武器规则
```go
// 当你使用轻型近战武器进行攻击动作时,可以用附赠动作攻击另一把轻型近战武器
// 条件:
// - 两把武器都必须是轻型
// - 副手攻击的伤害不加上属性调整值(除非是负数)
```

### 3. 擒抱规则
```go
// 使用力量(运动)检定对抗目标的力量(运动)或敏捷(体操)检定
// 条件:
// - 目标体型不能超过擒抱者一级
// - 擒抱者必须有一只空手
// 成功效果:
// - 目标速度变为0
// - 目标可以用动作进行逃脱检定
```

### 4. 推撞规则
```go
// 使用力量(运动)检定对抗目标的力量(运动)或敏捷(体操)检定
// 条件:
// - 目标体型不能超过推撞者一级
// 成功效果(二选一):
// - 目标倒地(Prone)
// - 目标被推开5尺
```

### 5. 掩护系统
```go
// 半身掩护: +2 AC和+2敏捷豁免
// 四分之三掩护: +5 AC和+5敏捷豁免
// 全身掩护: 无法被直接攻击
```

### 6. 专注豁免机制
```go
// 受到伤害时必须进行体质豁免
// DC = 10 或 伤害/2,取较高值
// 失败效果: 专注法术结束
```

### 7. 暴击规则
```go
// 自然20 = 暴击(自动命中)
// 暴击伤害 = 所有伤害骰x2 + 属性修正
// 自然1 = 大失败(自动未命中)
```

### 8. 跳跃系统
```go
// 跳远:
// - 有助跑(至少移动10尺): 力量值尺数
// - 立定跳远: 力量值/2尺数
// 跳高:
// - 有助跑: 3 + 力量调整值尺数
// - 立定跳高: (3 + 力量调整值)/2尺数
```

### 9. 坠落伤害
```go
// 每10尺坠落造成1d6伤害
// 最多20d6(200尺以上)
// 伤害类型: 钝击
```

### 10. 窒息规则
```go
// 闭气时间: 1 + 体质调整值(分钟),最少30秒
// 窒息后存活轮数: 体质调整值,最少1轮
// 窒息效果: 轮数用尽后生物昏迷
```

### 11. 团体检定
```go
// 当多个生物尝试一起完成某件事时使用
// 每个成员进行属性检定
// 如果至少一半成员成功,则团体成功
```

### 12. 合作检定
```go
// 当多个生物合作完成一件事时使用
// 领导者进行检定并具有优势(掷两次d20取高)
// 协助者必须能够执行该任务
```
